# Quote Shop.
## Simple shop project with Spring framework.

by Mambetkadyrov Salamatbek.

Deployment: https://quote-shop.herokuapp.com (HEROKU) <br>

Language: Java <br>
Version: 8 <br>
Spring version: 2.7 <br>
Spring dependencies: <br>
- Spring Web
- Spring Security 
- Spring JPA (PosgtreSQL) 
- Thymeleaf 
- Lombok
- Java Mail Sender
